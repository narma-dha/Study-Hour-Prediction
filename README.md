# Study Hour Prediction using Machine Learning
-------------------------------------------------------------------------------------
##  Overview
This project predicts student academic performance based on the number of study hours using Machine Learning techniques. It helps students understand how their study time can influence their marks.

------------------------------------------------------------------------------------------------------

##  Objective
- To analyze the relationship between study hours and marks  
- To build a predictive model using Linear Regression  
- To assist students in planning their study schedule effectively  

-----------------------------------------------------------------------------------------------------

##  Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Scikit-learn  

---------------------------------------------------------------------------------------------------------

##  Methodology
1. Data Collection (Study hours vs Marks dataset)  
2. Data Preprocessing and Cleaning  
3. Model Building using Linear Regression  
4. Training and Testing the Model  
5. Prediction and Visualization  

-----------------------------------------------------------------------------------------------------------------

##  Results
- The model shows a positive correlation between study hours and marks  
- As study hours increase, predicted marks also increase  
- The system provides accurate predictions for given input  

--------------------------------------------------------------------------------------------------------------------

##  Output
- User inputs study hours  
- System predicts expected marks  
- Graph visualization shows relationship between variables  

----------------------------------------------------------------------------------------------------------------

##  Future Scope
- Include additional factors like sleep, stress, and attendance  
- Improve accuracy using advanced ML models  
- Deploy as a web application for real-time use
